# Rock-Paper-Scissors

[Rock paper scissors](https://en.wikipedia.org/wiki/Rock_paper_scissors) is an intransitive hand game, usually played
between two people, in which each player simultaneously forms one
of three shapes with an outstretched hand. These shapes are
"rock" (a closed fist), "paper" (a flat hand), "scissors" (a fist
with the index finger and middle finger extended, forming V).

## How to run

1. Create a new virtual environment and install requirements from `requirements.txt`.
2. Run `python main.py --help`.

## How to test

The program is covered with Unit Tests.
Execute the command to run these tests: `pytest tests/`.
